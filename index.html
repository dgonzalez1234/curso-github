Crea un repositorio nuevo

Crea un directorio nuevo, ábre la consola y ejecuta

git init

para crear un nuevo repositorio de git.

git init - crea el staging y el repositorio remoto.

Clonar repositorio

Para clonar un repositorio, ábre la consola y ejecuta

git clone /path/to/repository

para descargar un repositorio de git.

Conectar repositorio remoto

Si no has clonado un repositorio ya existente y quieres conectar tu repositorio local a un repositorio remoto, ábre la consola y ejecuta

git remote add origin /path/to/repository

Ahora podrás subir tus cambios al repositorio remoto seleccionado.

para observar que se haya añadido correctamente, abre la consola y ejecuta

git remote -v

para borrar, abre la consola y ejecuta

git remote remove origin

Verificar rama

Para verificar la rama de mi repositorio, ábre la consola y ejecuta

git branch

para verificar la rama de mi repositorio de git.
